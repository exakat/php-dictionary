Wrapper
-------

A wrapper may be two things: 

+ a PHP wrapper, a built-in tool to process various URL-style protocols
+ a design pattern, also known as adaptor.

.. code-block:: php
   
   <?php
   
   // the PHP wrapper http
   $html = file_get_content('http://www.php.net/');
   
   ?>
