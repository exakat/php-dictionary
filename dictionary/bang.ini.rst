Bang !
------

``!`` is a character, used in various situations: 

+ not operator: ``var_dump(!1)`` is false
+ as a wildcard in regex
+ as a wildcard with ``fnmatch()``

.. code-block:: php
   
   <?php
   
       var_dump(!1);
   
   ?>
