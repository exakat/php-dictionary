Commit
------

A commit is a snapshot of a source code, at a specific point in time. It is a distinct feature of a version control system. 

It records changes made to the files in the project, such as dding, modifying, and deleting code. It has a unique identifier, called a hash, a message describing the changes, and metadata like the author and timestamp.

Several commits are grouped into a pull request, or PR, for code review.
