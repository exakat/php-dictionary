Phptoken
--------

``PHPToken`` is a class, part of the Tokenizer extension, that provides an object-oriented interface for working with tokens generated by PHP's tokenizer, used for parsing PHP code into tokens.

`Documentation <https://www.php.net/manual/en/class.phptoken.php>`__

See also `PHP 8.0: New PhpToken Tokenizer class <https://php.watch/versions/8.0/PhpToken>`_ and `Tokenize PHP Source Code using PhpToken in PHP 8.0 <https://lindevs.com/tokenize-php-source-code-using-phptoken-in-php-8-0>`_.
