Exponent
--------

Exponent is the mathematical operation to raise a number to the power of another number. 

There are two possibilities to use this operation : the `**` operator, and the pow() function.

.. code-block:: php
   
   <?php
   
   echo pow(-3, 2); // displays 9 
   
   echo 2 ** 3;    // displays 8
   ?>
