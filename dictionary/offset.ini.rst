.. _offset:

Offset
------

The term offset refers to the position or index of an element within an array or a string. Offsets are usually integers.

.. code-block:: php
   
   <?php
   
   $array = ['a', 'b', 'c'];
   $offset = 1;
   echo $array[$offset];  // b
   
   $string = 'ABC';
   $offset = 2;
   echo $string[$offset];  // C
   
   ?>
