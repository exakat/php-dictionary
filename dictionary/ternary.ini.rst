Ternary Operator
----------------

The expression ``(expr1) ? (expr2) : (expr3)`` evaluates to ``expr2`` if ``expr1`` evaluates to true, and ``expr3`` if ``expr1`` evaluates to false.

.. code-block:: php
   
   <?php
   
   $action = (empty($_POST['action'])) ? 'default' : $_POST['action'];
   
   ?>
