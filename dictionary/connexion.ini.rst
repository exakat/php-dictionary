Connexion
---------

Connexion refers to the communication link extablished between a running instance of PHP and a remote server.

The connexion may be established with different kind of servers: web, sql, nosql, ftp, etc. 

The connexion may be protected by a login and password, or other identification and authentication methods.

Once the connexion is established, PHP sends queries to the remote server, and receives results set.

The remote server may be on the same phyisical machine as PHP: it is considered remote as it acts autonomously.


See also https://www.php.net/manual/en/pdo.connections.php
