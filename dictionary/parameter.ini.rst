Parameter
---------

A parameter is a variable in a method definition. They have a name, and may have a default value and type. They may also be passed by reference, or by value (by default).

.. code-block:: php
   
   <?php
   
   function foo($parameter) {}
   
   $argument = 1;
   foo($argument);
   
   ?>
