Type Coverage
-------------

Type coverage reports how much of the PHP code has types. The elements that can be typed are: 

+ arguments in methods
+ return type
+ properties
+ class constants

Some elements cannot be typed: 
+ arrays
+ lexical variables
+ yield values
+ local variables.

.. code-block:: php
   
   <?php
   
   function foo($i = 0) : int {
       return $i + 1;
   }
   
   ?>
