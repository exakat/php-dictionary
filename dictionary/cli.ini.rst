Command Line Interface (CLI)
----------------------------

PHP supports a CLI mode : command line interface. It processes commands in the form of lines of text. 

.. code-block:: php
   
   <?php
   
   print_r($argv);
   
   ?>
   


`Documentation <https://www.php.net/manual/en/features.commandline.php>`__
