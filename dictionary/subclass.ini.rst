Subclass
--------

A ``subclass`` is a class that extends another class. It is also called a child of that class.

Subclasses have the same features than any other class, besides the extension.

.. code-block:: php
   
   <?php
   
   class A {}
   
   // this is a subclass
   class B extends A {}
   
   ?>
