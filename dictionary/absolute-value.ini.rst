Absolute Value
--------------

An ``absolute`` value is the positive value of a real number.

There exists the native function ``abs()`` to calculate this value.

.. code-block:: php
   
   <?php
   
   $a = 10;
   echo abs($a); // 10
   
   $b = -12;
   echo abs($b); // 12
   
   ?>
