Branch
------

A branch is one of the two alternative in a if then expression. There is the ``then`` branch, and the ``else`` branch. 

.. code-block:: php
   
   <?php
   
   // a match with 2 arm, including the default
   if ($a) {
   	$branch = 'then';
   } else {
   	$branch = 'else';
   }
   
   ?>
