Associative Array
-----------------

An associative array is an array whose keys are strings. There may be integer index, mixed with the string ones.

.. code-block:: php
   
   
   <?php
   
   $associative = ['a' => 1, 'b' => 4];
   
   echo $associative['a'];
   
   ?>
   


`Documentation <https://www.php.net/manual/en/intro.array.php>`__
