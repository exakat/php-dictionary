Cross Site Request Forgery (CSRF)
---------------------------------

a CSRF attack happens when a malicious website tricks a logged-in user’s browser into sending an unwanted request to another application. CSRF requires both a intermediate website and a victim website.

`Documentation <https://en.wikipedia.org/wiki/Cross-site_request_forgery>`__

See also https://brightsec.com/blog/csrf-vs-xss/, https://owasp.org/www-community/attacks/csrf
