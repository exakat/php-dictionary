phpinfo()
---------

``phpinfo()`` is one of the most famous PHP function. It outputs information about PHP's configuration and compilation, in HTML format.

.. code-block:: php
   
   <?php
   
       // Show all information, defaults to INFO_ALL
       phpinfo();
   
   ?>
