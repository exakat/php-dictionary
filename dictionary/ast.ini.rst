Abstract Syntactic Tree
-----------------------

An abstract syntax tree (AST), is a tree representation of the abstract syntactic structure of PHP source code. 

`Documentation <https://en.wikipedia.org/wiki/Abstract_syntax_tree>`__

Added in PHP 7.0
