Echo Tag
--------

Echo tag is a special PHP opening tag : ``<?=``. It combines the opening tag, and a call to the echo function.

The echo tag should not be confused with PHP short tags ``<?``.

It is also called the short echo tag.


.. code-block:: php
   
   <?php $a = 1; ?><?= $a; ?>
