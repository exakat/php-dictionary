Data Container
--------------

A data container is a structure that stores data. It is a generic concept, that covers variables, properties and arrays. 

.. code-block:: php
   
   <?php
   
   $a = 1;
   $object->property = 'b';
   $array = ['a', 3, 4.4];
   
   ?>
