Bracket
-------

Bracket is a generic class of characters. 

They include 

+ Parenthesis, ``()``, or round brackets
+ Curly ``{}``, or accolades
+ Square ``[]``, or brackets

Brackets are a element of the PHP syntax.


.. code-block:: php
   
   <?php
   
       // curly brackes for block
       {
           // square brackets for array syntax
           $array[1] = 2;
       }
   
   ?>
